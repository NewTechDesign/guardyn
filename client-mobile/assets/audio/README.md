# Audio Assets for Guardyn Calls

This directory contains audio files used for call notifications.

## Required Files

Place the following audio files in this directory:

| File | Description | Recommended Duration |
|------|-------------|---------------------|
| `ringtone_incoming.mp3` | Incoming call ringtone | 30s (loops) |
| `dial_tone.mp3` | Outgoing call dial tone | 10s (loops) |
| `call_connected.mp3` | Call connected notification | 0.5-1s |
| `call_ended.mp3` | Call ended notification | 0.5-1s |
| `busy_tone.mp3` | Busy signal | 3s |

## Recommendations

- Use royalty-free audio or create custom tones
- Optimize for mobile (128kbps MP3 is sufficient)
- Keep files small (under 500KB each)
- Test on both iOS and Android devices

## Sources for Free Audio

- [Freesound.org](https://freesound.org) - Creative Commons audio
- [Mixkit](https://mixkit.co/free-sound-effects/) - Free sound effects
- Create custom tones using audio software
